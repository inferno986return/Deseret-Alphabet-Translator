Deseret Alphabet Translator
REST API

Resource URL

http://2deseret.com/json/translation

Parameters

english
English to be translated into Deseret Alphabet
deseret
Deseret Alphabet to be translated into English

If both parameters are provided, only english will be used. Deseret Alphabet characters should be in Unicode, or in the ASCII representation provided in blue letters on the Deseret Alphabet keyboard (as supported by the AdamicBee typeface):

  • Example: Deseret Alphabet to English

    POST
http://2deseret.com/json/translation
{"english": "Learning the Deseret Alphabet is fun."}
    {
    "deseret": "𐐢𐐲𐑉𐑌𐐮𐑍 𐑄 𐐔𐐯𐑅𐐨𐑉𐐯𐐻 𐐈𐑊𐑁𐐰𐐺𐐯𐐻 𐐮𐑆 𐑁𐐲𐑌."
}

  • Example: Deseret Alphabet (Unicode) to English.

    POST
http://2deseret.com/json/translation
{"deseret": "𐐢𐐲𐑉𐑌𐐮𐑍 𐑄 𐐔𐐯𐑅𐐨𐑉𐐯𐐻 𐐈𐑊𐑁𐐰𐐺𐐯𐐻 𐐮𐑆 𐑁𐐲𐑌."}
    {
    "english": "Learning the Deseret Alphabet is fun."
}

  • Example: Deseret Alphabet (ASCII) to English.

    POST
http://2deseret.com/json/translation
{"deseret": "L-rn@< \ D`sir`t &lf^b`t @z f-n."}
    {
    "english": "Learning the Deseret Alphabet is fun."
}
